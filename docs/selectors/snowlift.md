# snowlift.yml



## snowlift


The snowlift is that screen that you view image in a lightbox format.

Facebook calls it a "fbPhotoSnowlift", hence the name "snowlift."
A more general name of it is "Facebook Photo Theater."


__Selectors:__

 * .fbPhotoSnowlift
